# Shepherds' Stories
A tool for missionaries to streamline communication with supporters.

---

## Introduction
There are currently about 430,000 full-time Christian missionaries worldwide, and nearly half will serve for less than 5 years. 
Most of the time, missionaries quit due to financial support issues, a lack of communication from home, and homesickness
(Nations Outreach, 2025).  

The primary problem this application will attempt to address is the friction in communication between the missionary and 
the lack of explicit encouragement from their supporters.

The missionary will be able to make posts as if they were posting on a social media site for quick updates, 
or record videos to more easily convey their emotions through tone and body language.

Supporters will be able to see the missionary, get a quick summary of what has been going on, and encourage them through comments on any updates.

---

## MVP Features
- A missionary's content can only be seen by a supporter through registration of the missionary's reference number
- Secure user authentication for missionary and supporter accounts 
- Role-based access control separates missionary and supporter features 
- Missionary dashboard for viewing core application data 
- Supporter contact entry management with view and delete functionality 
- Supporter account registration through a valid reference number
- Missionary update management with create, view, edit, and delete functionality 
- Supporter-facing portal for viewing published missionary updates 
- Persistent database storage for users, supporters, invitations, and updates 
- Input validation and error handling for forms and account flows
- Technical and user documentation
